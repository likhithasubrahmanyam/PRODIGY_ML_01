# PRODIGY_ML_01
# 🏠 House Price Prediction using Linear Regression

> A Machine Learning project that predicts house prices based on living area, number of bedrooms, and number of bathrooms using the Linear Regression algorithm.

---

## 📖 Project Overview

The objective of this project is to build a Linear Regression model capable of predicting house prices using important housing features. This project demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, prediction, and evaluation.

---

## 🎯 Objective

Develop a regression model to estimate house prices using:

- 📐 Living Area (`GrLivArea`)
- 🛏️ Number of Bedrooms (`BedroomAbvGr`)
- 🛁 Number of Bathrooms (`FullBath`)

---

## 📂 Dataset

**Dataset:** House Prices – Advanced Regression Techniques

Source: Kaggle

The dataset contains information about residential houses and their corresponding sale prices.

---

- Python

## 📊 Machine Learning Workflow

- Import required libraries
- Load and inspect the dataset
- Select important features
- Handle missing values (if any)
- Split data into training and testing sets
- Train the Linear Regression model
- Predict house prices
- Evaluate model performance
- Predict the price of a new house

---

## 📈 Model Evaluation

The model was evaluated using:

- ✅ Mean Absolute Error (MAE)
- ✅ Mean Squared Error (MSE)
- ✅ Root Mean Squared Error (RMSE)
- ✅ R² Score

These metrics help measure the prediction accuracy of the model.
